Bu yöntem İle bulunan Proxilerin Kullanımıyla herhangi bir ilgim yoktur.
Bütün Telif hakları Şahsım Olan H04x'a Aittir!
-----------------------------------------------
    TR           Kullanım
----------------          ---------------------
- Proxy.txt Adındaki Dosyanın içine Check Edilecek Proxy'leri yazın.
- Daha Sonra Klasör içinde 'CMD' Açın Ve > "pip install -r requirements.txt" Yazıp Enter Basın.
- Şimdi ise Tekrar 'CMD' Açıp içine > "python h04xproxies.py" yazıyoruz,
- Ve Karşınıza Dosya Girme Yeri çıkıcak Oraya ise 'Proxy.txt' Yazıp Bekliyoruz Ve Check Etmeye Başlıyor.

-----------------------------------------------
    EN            Usage
-----------------      ------------------------        
- Write Proxies to Check in the File Named Proxy.txt.
- Then Open 'CMD' in the folder and > Type "pip install -r requirements.txt" and press Enter.
- Now open 'CMD' again and write > "python h04xproxies.py" in it,
- And the File Entering Place will appear. We write 'Proxy.txt' and wait and it starts to check.
